# Measure Converter  ⚖️
The measures conversion app will allow your users to select a measure – metric or imperial
– and convert it to another measure.

# Command To Install

### Install dependencies

```sh
flutter pub get
```

### Run application

```sh
flutter emulators --launch <emulator_name>
flutter run
```
